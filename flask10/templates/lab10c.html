Lab 10C: The Flask Framework

The Framework

For this lab, I decided to use a framework called Flask; a lightweight python-based web-application framework. I decided to use this framework given its ease of use and compatibility with MacOS. Please note that all command lines were executed in terminal as I was using a MacOS system for this section of the lab. Also, each of the commands were executed as written; the name of my flask folder was flask10.

Framework Installation

Flask installation is very simple as it only need be installed from the command line.

  1. Open Terminal
  2. Install Python
    1. brew install python3
  3. Create a folder and navigate into it
    1. mkdir flask10
    2. cd flask10
  4. Install Flask in the current folder
    1. pip install flask
  5. Create an __init__.py file
    1. touch __init__.py
    2. vim __init__.py
    3. Copy and paste the template into the __init__.py file

Building the Page

Once Flask has been installed and the init file has been created, the template and static files for the program can be constructed. Keep in mind that all template and static files will form the content for the webpage.

  1. In the current directory, create a folder called templates'
  2. Navigate to the folder called templates
  3. Inside the templates folder, construct an HTML file
  4. Go back to the original flask directory, create a folder called static
  5. Inside the static folder, construct a CSS file
  6. Connect the CSS to the HTML using the link tag in the head of the HTML file
  7. Link the HTML file to the __init__.py file
  8. Run __init__.py to view your webpage
    1. flask --app flask10 run --debug
